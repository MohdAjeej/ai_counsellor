Hero image (landing page)
========================
The hero uses a real photograph from Pexels (student/study theme). If that image does not load (e.g. network blocks it), add your own real photo here:

  hero.jpg

Use a real photograph of a student studying, graduation, or study-abroad scene (not AI-generated). For example, download a free stock photo from Pexels or Unsplash and save it as hero.jpg in this folder. The hero will then show your photo.
